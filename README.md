# PhotoBooth V2 — Releases

Repositório público de distribuição do PhotoBooth V2.

Este repositório contém somente manifestos e arquivos compilados publicados em [Releases](https://github.com/Letacio-Bezerra/photobooth-v2-releases/releases). O código-fonte permanece privado.

- `app-manifest.json`: versão mais recente do aplicativo.
- `templates-manifest.json`: catálogo de templates.
- Releases: instaladores, pacotes de atualização e ZIPs de templates.

Os manifestos são atualizados somente depois que os arquivos correspondentes são publicados e validados.
